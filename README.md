# Ping pong
## Как пользоваться
```powershell
python .\main.py
 ```
## Описание
Данный проект представляет собой реализацию классической аркадной игры Pong с использованием библиотеки Pygame. Здесь два игрока могут соревноваться на одном компьютере. Левая ракетка управляется клавишами W/S, правая — стрелками вверх/вниз. Мяч отскакивает от стенок и ракеток, а пропустивший соперник теряет очко. Игра продолжается до достижения одним из игроков 7 очков.
